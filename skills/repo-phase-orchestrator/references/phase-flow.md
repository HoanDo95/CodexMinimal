# Phase Flow

1. Read approved spec.
2. Write or refresh phase plan.
3. Write or refresh tracker.
4. Refresh artifact registry and current-work state.
5. Determine current phase.
6. Handoff the current phase to Codex CLI native execution by default, or to another selected adapter when policy requires it.
7. Collect verification status from execution.
8. Update tracker and telemetry.
9. Stop before the next phase until status is clean.
